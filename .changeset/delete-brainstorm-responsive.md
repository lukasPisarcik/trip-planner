Trip management & UX fixes:

- **Delete a trip** — from the sidebar (hover trash icon) or the All Trips cards, behind a confirmation dialog. Deleting the trip you're viewing returns you to the trip list.
- **Brainstorm tab** — a new first tab: a free-text, auto-saving scratchpad for ideas, links, and findings. The AI co-pilot reads it as context (and never overwrites it).
- **AI panel responsivity** — the assistant panel now overlays as a dismissable drawer on narrow screens (with a backdrop) instead of squishing the content; it only reserves side space on wide (≥1024px) screens.
- Removed the inline click-to-edit Hero fields (title/dates/tagline); user input now lives in the Brainstorm tab.
